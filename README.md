# Scatter-ble 장치
## 2021 단국대 실무중심산학협력프로젝트



## 개요

#### 코로나 예방을 목적으로 하는, 다중이용시설 사용자 현황에 대한 정보 확인 간평화 및 편리성 도모

## 구조

#### Web: javascript

#### Android:java  :  https://github.com/gudwnsepdy/Scatter_iOS

#### Ios: swift  :  https://github.com/gudwnsepdy/Scatter_Android
